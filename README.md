# Flutter Stopwatch App

A simple stopwatch mobile application built with Flutter.

## Description

This app implements a basic stopwatch with start, stop, and reset functionality.
It displays the elapsed time in minutes, seconds, and milliseconds with a circular progress indicator.

## Features

- Start, stop, and reset controls
- Real-time time tracking
- Circular progress indicator
- Clean and modern dark UI
- Built using Flutter state management

## Technologies

- Flutter
- Dart
- Material Design
